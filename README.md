# 🩺 Cancer Survival Prediction System

## 📌 Project Overview

- The Cancer Survival Prediction System is a Machine Learning-based web application that predicts cancer survival outcomes using patient demographic, medical, and treatment-related information.
- The project utilizes a Decision Tree Classifier trained on cancer patient data and provides real-time predictions through an interactive Streamlit web interface.
- This application can assist healthcare researchers and medical professionals in analyzing patient survival patterns and supporting decision-making processes.
